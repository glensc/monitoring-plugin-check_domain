Nagios/Icinga Plugin: check\_domain
===================================

Nagios/Icinga plugin for checking a domain name expiration date

Usage:
```
$ ./check_domain.sh -d github.io
OK - Domain github.io will expire in 43 days (2014-03-08).
```
